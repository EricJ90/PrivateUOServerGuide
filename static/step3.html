Step 3: Opening and building ClassicUO with Git

  1. Unzip the ClassicUO files to the directory you want.
  2. Open Git in the folder you installed your ClassicUO
  3. Force a dependency update using git submodule update --init --recursive
  4. Rebuild the project
  5. Update the settings.json file to have the file locations for your UO client.
  6. Click Run and log in with your admin account!
  7. That's all. Enjoy your private server!
