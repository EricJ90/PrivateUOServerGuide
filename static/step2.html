Step 2: Preparing RunUO

  1. After you installed the RAR extractor, use it on the RunUO package
  2. Extract the files to the directory you want RunUO to live
  3. Once extracted, open the RunUO.exe file.
  4. This will ask you to create a username and password for the master account. Enter these details.
  5. Once you create the account, RunUO will open a port to listen for incoming signals. This server and port are listed last on the list
  6. It will look like:
  7. Remember the last IP address and port. In the picture above, it is 127.0.0.1: port 2593
  8. Leave this window open. It will keep the server live while you play.
