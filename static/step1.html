A guide for noobs

Step 1: Preparing your computer

  1. Download all of the software and install then on your computer with the default settings linked on the left side of this page: Home
  2. Install the Ultima Online Classic Client to the default location. Run it to start patching. This may take a while, the software is downloading files it needs to render the game. Close when the patch is done (play button is clickable).
  3. Install Visual Studio with the default settings and choose ASP.NET and web development, .NET Desktop Development, and Universal Windows Platform when you arrive at the workloads page. Once that has installed, close Visual studio until later.
  4. Install the RAR extractor with default settings.
  5. Don't worry about the RunUO.rar file just yet. We'll get to that in Step 2
  6. You now have the proper software to run your own local UO server!
